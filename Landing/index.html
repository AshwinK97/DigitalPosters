Scaling Levels of Detail

When building a poster using our editor, all data can be added using blocks. The data that is presented will be optimized for the presentation medium. This means that large displays will show less information and will focus more on images.

Frequently Asked Questions

  • Can I export any posters that I created?

    Yes, you can export your finished poster as a PDF which can be viewed on any pdf reader or browser.

  • What happens if I can't finish my poster in one sitting?

    There is a save feature that allows you to save your poster whether you are finished or not. Then you can access your saved posters through your main dashboard.

  • Does the text editor have enough features?

    Of course! we have a wide selection of features. Text formatting, picture editing, publishing are all just few of the many great features Poster Cloud provides.

  • How do the block modules work when creating posters?

    Users can add and remove blocks as they prefer. Blocks can be used to house images, text or headings. The block system allows each poster to be modular and easy to change.
